# 🛍️ منصة التجارة الإلكترونية المتكاملة - Laravel 10

منصة تجارة إلكترونية شاملة ومتطورة مبنية بإطار Laravel 10، مصممة للأعمال التجارية الحديثة بواجهة جميلة ومتجاوبة ووظائف قوية.

## 🌟 المميزات الرئيسية

### 🛒 التجارة الإلكترونية الأساسية
- **إدارة المنتجات**: عمليات CRUD كاملة مع الفئات والعلامات التجارية والمتغيرات
- **سلة التسوق**: نظام متقدم مع إدارة الكميات واستمرارية الجلسة
- **معالجة الطلبات**: دورة حياة الطلب الكاملة من الدفع إلى تتبع التوصيل
- **التكامل مع الدفع**: PayPal مع معالجة آمنة للمدفوعات
- **نظام الكوبونات**: أكواد الخصم والحملات الترويجية
- **قائمة الرغبات**: حفظ المنتجات المفضلة للشراء لاحقاً

### 👥 إدارة المستخدمين
- **نظام الأدوار المتعددة**: مشرف، مستخدم، وضيف
- **المصادقة الاجتماعية**: تسجيل الدخول عبر Facebook، Google، وGitHub
- **ملفات التعريف**: لوحات تحكم شخصية مع سجل الطلبات
- **التقييمات والمراجعات**: نظام ملاحظات العملاء مع تقييم النجوم

### 📝 إدارة المحتوى
- **نظام المدونات**: مدونة متكاملة مع الفئات والعلامات والتعليقات
- **إدارة البانرات**: بانرات ديناميكية ومنزلقات ترويجية
- **صديق لمحركات البحث**: عناوين URL محسنة وعلامات وصفية
- **جاهز للغات متعددة**: منظم للترجمة السهلة

### 🎨 واجهة مستخدم حديثة
- **تصميم متجاوب**: نهج يعطي الأولوية للجوال مع Bootstrap 4
- **مخطط ألوان حديث**: لوحة من البنفسجي العميق، الفيروزي، والأصفر الخردلي
- **تكامل وسائل التواصل الاجتماعي**: أيقونات اجتماعية مخصصة
- **رسوم متحركة سلسة**: انتقالات CSS وتفاعلات دقيقة
- **دعم الوضع الليلي**: محسن لجميع تفضيلات المشاهدة

### 📊 لوحة تحكم المشرف
- **لوحة التحليلات**: إحصائيات المبيعات والطلبات والعملاء
- **إدارة المخزون**: تتبع المخزون والتنبيهات
- **إدارة الطلبات**: سير عمل معالجة الطلبات الكامل

## 🛠️ دليل التثبيت

### 1. استنساخ المستودع
```bash
git remote add origin https://github.com/OsamaFalahi/novacart-ecommerce-laravel.git
cd complete-ecommerce-laravel-10
```

### 2. تثبيت الاعتماديات
```bash
composer install
npm install
```

### 3. تكوين البيئة
```bash
cp .env.example .env
php artisan key:generate
```

### 4. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات وتحديث .env ببيانات الاعتماد
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ecommerce
DB_USERNAME=your_username
DB_PASSWORD=your_password
```

### 5. تشغيل الترحيل والبيانات
```bash
php artisan migrate
php artisan db:seed
```

### 6. ربط التخزين وتجميع الأصول
```bash
php artisan storage:link
npm run build
```

### 7. تشغيل خادم التطوير
```bash
php artisan serve
```

قم بزيارة `http://localhost:8000` للوصول إلى منصة التجارة الإلكترونية الخاصة بك.

## 🔧 التكوين

### إعداد الدفع
1. تكوين بيانات اعتماد PayPal في `.env`:
```env
PAYPAL_MODE=sandbox
PAYPAL_SANDBOX_CLIENT_ID=your_sandbox_client_id
PAYPAL_SANDBOX_CLIENT_SECRET=your_sandbox_client_secret
PAYPAL_LIVE_CLIENT_ID=your_live_client_id
PAYPAL_LIVE_CLIENT_SECRET=your_live_client_secret
```

### إعداد تسجيل الدخول الاجتماعي
1. إضافة بيانات اعتماد OAuth في `config/services.php`:
```php
'facebook' => [
    'client_id' => env('FACEBOOK_CLIENT_ID'),
    'client_secret' => env('FACEBOOK_CLIENT_SECRET'),
    'redirect' => env('FACEBOOK_REDIRECT_URI'),
],
'google' => [
    'client_id' => env('GOOGLE_CLIENT_ID'),
    'client_secret' => env('GOOGLE_CLIENT_SECRET'),
    'redirect' => env('GOOGLE_REDIRECT_URI'),
],
```

### تكوين البريد الإلكتروني
```env
MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=your_username
MAIL_PASSWORD=your_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=hello@example.com
MAIL_FROM_NAME="${APP_NAME}"
```
